# 🎨 Анализатор доминирующего цвета изображения

Интерактивное desktop-приложение на Python с графическим интерфейсом для определения доминирующих цветов в изображениях с использованием алгоритма K-means кластеризации.

## ✨ Возможности

- **Загрузка изображений** - поддерживаются форматы JPG, JPEG, PNG, BMP, GIF, TIFF
- **Алгоритм K-means** - определение доминирующих цветов методом кластеризации
- **Настраиваемый анализ** - выбор количества цветов для анализа (от 2 до 10)
- **Визуализация результатов**:
  - Доминирующий цвет с RGB-значениями
  - Палитра всех найденных цветов с процентным соотношением
  - Интерактивные графики с использованием Matplotlib
- **Экспорт результатов** - сохранение изображения с наложенной палитрой цветов
- **Асинхронная обработка** - анализ изображений выполняется в отдельном потоке без блокировки интерфейса

## 🛠️ Технологии

- **Python 3.x**
- **OpenCV** (cv2) - обработка изображений и кластеризация
- **Pillow** (PIL) - работа с изображениями в Tkinter
- **Matplotlib** - визуализация цветовой палитры
- **NumPy** - эффективная работа с массивами данных
- **Tkinter** - графический интерфейс пользователя
